# My Grocery List

Personal grocery shopping app — single HTML file, no install needed.

## Live App
🔗 https://YOUR-USERNAME.github.io/grocery/grocery-list.html

Open in Safari on iPhone → Share → Add to Home Screen

## Features
- 80+ Indian grocery items pre-loaded across 11 categories
- Tap items to select for current shopping trip
- Sort selected items to the top
- Filter by category or search by name
- Add / delete items
- Reset to full default list anytime
- Works offline after first load

## How to update the list
1. Edit grocery-list.html in VS Code
2. Upload the new file to this repo (drag & drop on GitHub)
3. Changes go live in ~1 minute

## Tech
Plain HTML + CSS + JavaScript. No frameworks, no dependencies.
